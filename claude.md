# Claude AI Workspace Rules

- **Project:** OpenClaw SecOps Red Team
- **Root Wiki:** `[[LLM-WIKI/000_Home.md]]`
- **SOP (Standard Operating Procedures):** 
  1. เมื่อเริ่มต้นการสนทนาใหม่ หรือไม่มั่นใจบริบท ให้โหลดข้อมูลจากโฟลเดอร์ `LLM-WIKI/` เป็นจุดตั้งต้นเสมอ
  2. โปรเจกต์นี้อิงสถาปัตยกรรม OpenClaw, Telegram Bots ควบคุมการทำงานของ Agent 5 ตัวบน Kali Linux
  3. หากเป้าหมายคือการปรับปรุงโค้ดหรือแก้โครงสร้าง **คุณต้องทำการแก้ไฟล์อธิบายระบบที่เกี่ยวข้องภายในโฟลเดอร์ `LLM-WIKI/` ควบคู่ไปด้วยเสมอ** (Self-Maintaining Wiki)
  4. ทุกไฟล์ในโปรเจกต์จะเชื่อมโยงถึงกันด้วยแพทเทิร์นของ Obsidian Markdown `[[wikilinks]]` โปรดใช้รูปแบบนี้เสมอถ้ามีการอ้างอิงระหว่างไฟล์
